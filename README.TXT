Don't touch the data folder.
If you don't have java installed download it from here: https://www.java.com/download/ie_manual.jsp

To use, double click gaiaScraper.
In the Account number field, please input AND SAVE your gaiaonline account number.
You can find this number either on your account page or archive page

Example:
	https://www.gaiaonline.com/journal/?mode=archive&u=XXXXXXXX
	https://www.gaiaonline.com/profiles/username/XXXXXXXX/

Where XXXXXXXX is account number.

Click Start to begin the archival process. You'll know it's running if,
after a moment, A folder named with the account number appears. Inside that 
is a folder named "Diary", and that is where your entries are.
When the program is finished, a popup will tell you.

One possible reason for nothing being archived is that the user's journal entries
are private.

If you have any issues or inquiries, feel free to contact me.

Enjoy!
